# WebGAL_Terre
## 立即体验

##### 下载链接( Windows )

https://github.com/MakinoharaShoko/WebGAL_Terre/releases

##### MacOS 和 Linux 用户可以直接下载源代码，然后使用 Node.Js 运行 

```
npm install -g pnpm
pnpm install
node index.js
```

## 使用说明

#### 1、游戏配置

可以配置游戏信息与标题界面

#### 2、资源管理

可以上传并管理资源

#### 3、场景编辑

注意，整个游戏的入口场景是 start，你可以使用分支跳转或者切换场景来进入其他的场景。

在左侧是场景列表，在右侧是场景编辑器，你可以通过点击 “添加语句” 按钮来开始制作你的 GalGame！
